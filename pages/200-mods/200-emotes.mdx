# Эмоции

![](/assets/mods/emotes/demo.webp)

Оживи своего персонажа с помощью анимированных эмоций.

Другие игроки увидят твои эмоции только если у них установлен тот же мод.

<FatLink href="https://modrinth.com/mod/emotecraft">
    Скачать EmoteCraft
</FatLink>

## Как использовать

![](/assets/mods/emotes/howto.webp)

1. Нажми клавишу **B**, чтобы открыть меню эмоций
2. Для настройки нажми **Все эмоции**, затем **Настройка эмоций**
3. Выбери эмоцию и место для неё в круге быстрого доступа
4. Нажми **Готово**, затем снова **B**, чтобы проверить настроенные эмоции
