# Расследования

Доступ к CoreProtect и AuxProtect получают Интерпол, Хранители, некоторые работники суда и Хелперы. Инструменты предназначены для расследования краж и других нарушений

Для удобной визуализации есть клиентский мод [Watson](https://github.com/digitaldrugstech/WatsonMod) — наш форк с поддержкой 1.21.11 и 26.1, AuxProtect и оптимизацией

<CustomBlock type="warn">

#### Все запросы логируются

Эти инструменты только для расследований. Не используй их чтобы проверять где гуляет друг или что лежит в чужих сундуках просто так. Мы видим все запросы и можем снять доступ

</CustomBlock>

## CoreProtect

Логирует блоки, сундуки, инвентари, убийства, чат, команды

### Инспекция

`/co inspect` или `/co i` — включает режим инспекции. Кликай по любому блоку или сундуку чтобы увидеть историю действий

`/co near` — история действий в радиусе 5 блоков

### Поиск

`/co lookup` или `/co l` — поиск по логам

**Параметры:**

| Параметр | Пример | Описание |
|----------|--------|----------|
| `u:<ник>` | `u:haffk` | Игрок (несколько через запятую) |
| `t:<время>` | `t:3d` | Период (w/d/h/m/s, можно комбинировать: `t:1d6h`) |
| `r:<радиус>` | `r:10` | Радиус в блоках |
| `a:<действие>` | `a:container` | Тип действия |
| `i:<блок>` | `i:diamond_ore` | Только определённый блок/предмет |
| `e:<исключить>` | `e:stone,dirt` | Исключить блок/предмет |

Диапазон времени: `t:1h-2h` — от часа до двух назад

Неигровые источники: `u:#fire`, `u:#tnt`, `u:#creeper`, `u:#explosion`

Глобальный поиск: `r:#global`

**Флаги:** `#count` (количество записей)

### Действия

| Действие | Что логируется |
|----------|---------------|
| `a:block` | Блоки (установка и разрушение) |
| `a:+block` / `a:-block` | Только установка / только разрушение |
| `a:container` | Сундуки, бочки, печки |
| `a:+container` / `a:-container` | Только положили / только взяли |
| `a:inventory` | Инвентарь игроков |
| `a:item` | Предметы (дроп, подбор) |
| `a:kill` | Убийства мобов и игроков |
| `a:chat` | Чат |
| `a:command` | Команды |
| `a:click` | Взаимодействия |
| `a:sign` | Текст на табличках |

### Примеры и кейсы

<Collapse summary="Кража из сундука">

Игрок жалуется что у него пропали алмазы из сундука

1. Встань рядом с сундуком
2. `/co i` — включи инспекцию
3. Кликни по сундуку — увидишь кто и когда брал предметы
4. Или `/co l r:3 t:7d a:container` — все взаимодействия с контейнерами рядом за неделю

</Collapse>

<Collapse summary="Разрушение постройки">

Кто-то сломал чужие блоки

1. `/co i` — кликай по разрушенным блокам
2. Или `/co l r:20 t:3d a:-block` — кто ломал блоки в радиусе 20 за 3 дня
3. Для конкретного игрока: `/co l u:подозреваемый r:20 t:3d a:-block`

</Collapse>

<Collapse summary="Поджог">

Следы огня на постройке

`/co l r:15 t:1d u:#fire` — проверить что горело

`/co l r:15 t:1d i:flint_and_steel` — кто использовал зажигалку рядом

</Collapse>

<Collapse summary="Убийство">

Кого убил игрок или кто убил игрока

`/co l u:подозреваемый t:1d a:kill` — все убийства игрока за день

</Collapse>

<Collapse summary="Что писал в чат">

`/co l u:подозреваемый t:1d a:chat` — все сообщения игрока за день

</Collapse>

## AuxProtect

Дополнение к CoreProtect. Логирует позиции игроков, инвентари целиком, телепортации, урон

### Команды

| Команда | Описание |
|---------|----------|
| `/ap lookup` или `/ap l` | Поиск по логам AuxProtect |
| `/ap inventory <ник>` | История инвентарей игрока |
| `/ap playtime <ник>` | Визуальный график активности за 14 дней |

### Инвентари

`/ap inventory <ник>` — показывает список снимков инвентаря игрока за последние 2 недели. Кликни по записи чтобы открыть содержимое инвентаря на тот момент

Полезно когда игрок утверждает что у него были определённые вещи, или для проверки подозрительного обогащения

### Параметры поиска

Синтаксис похож на CoreProtect но с другими именами:

| Параметр | Описание |
|----------|----------|
| `user:<ник>` | Игрок |
| `time:<время>` | Период |
| `action:<действие>` | Тип действия |
| `target:<цель>` | Предмет, моб или игрок (поддерживает `*`) |
| `radius:<число>` | Радиус |
| `world:<мир>` | Мир |

### Действия AuxProtect

| Действие | Описание |
|----------|----------|
| `pos` | Позиции игрока |
| `tp` | Телепортации |
| `hurt` | Получение урона |
| `kill` | Убийства |
| `inventory` | Снимки инвентарей |
| `command` | Команды |
| `chat` | Чат |
| `drop` / `pickup` | Дроп и подбор предметов |

### Примеры

`/ap l user:haffk action:pos time:1h` — где находился haffk за последний час

`/ap l user:haffk action:drop time:1d target:diamond` — когда haffk выбрасывал алмазы

`/ap l action:pickup radius:10 time:30m` — кто подбирал предметы рядом за 30 минут

## Watson

Клиентский мод, визуализирует результаты CoreProtect в 3D прямо в игре. Наш форк мода Watson с поддержкой 1.21.11 и 26.1, оптимизацией и поддержкой AuxProtect

<FatLink href="https://github.com/digitaldrugstech/WatsonMod">Скачать Watson (PRDX Fork)</FatLink>

Требует: Fabric + Malilib

Что добавлено в форке:
- Поддержка 1.21.11 и 26.1
- Поддержка AuxProtect
- Оптимизация производительности

### Основные команды

| Команда | Описание |
|---------|----------|
| `/w display` | Вкл/выкл 3D отображение |
| `/w clear` | Очистить данные |
| `/w vector` | Вкл/выкл линии между действиями |
| `/w filter add <ник>` | Фильтр по игроку |

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| :kbd[L] + :kbd[C] | Открыть GUI |
| :kbd[↑] / :kbd[↓] | Навигация по действиям |
