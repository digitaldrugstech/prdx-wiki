# Творческий сервер

Творческий сервер — место для тестирования построек и планирования проектов перед реализацией на основном сервере. У каждого игрока есть личный участок размером 100x100 блоков, где можно строить что угодно.

## Команды для управления участками

Основные команды:
- `/plot auto` — получить случайный свободный участок
- `/plot home` — вернуться на свой участок 
- `/plot clear` — очистить свой участок

### Создание участка и настройка доступа

Чтобы получить участок, напиши `/plot auto`. Для возвращения на участок используй `/plot home`.

Настройка доступа:
- `/plot trust <ник>` — разрешить игроку строить на твоем участке
- `/plot deny <ник>` — запретить игроку доступ на участок
- `/plot helpers` — посмотреть, кому разрешено строить на твоем участке
- `/plot denied` — посмотреть список игроков, которым запрещен доступ

### Дополнительные команды PlotSquared

- `/plot middle` — телепортироваться в центр участка
- `/plot tp <X;Y>` — телепортироваться на участок по координатам
- `/plot visit <ник>` — посетить участок другого игрока
- `/plot merge` — объединить соседние участки (если они принадлежат тебе)
- `/plot unlink` — разделить объединенные участки

## WorldEdit

На творческом сервере доступен WorldEdit — мощный инструмент для быстрого редактирования больших областей.

### Основные команды WorldEdit

#### Выделение области

- `//wand` — получить инструмент выделения (деревянный топор)
- `//pos1` и `//pos2` — установить первую и вторую точку выделения
- `//set <блок>` — заменить все блоки в выделенной области
- `//replace <блок1> <блок2>` — заменить один тип блоков на другой

#### Копирование и вставка

- `//copy` — копировать выделенную область
- `//paste` — вставить скопированную область
- `//rotate <градусы>` — повернуть скопированную область
- `//flip <направление>` — отразить скопированную область

#### Генерация и заполнение

- `//sphere <блок> <радиус>` — создать сферу
- `//cyl <блок> <радиус> <высота>` — создать цилиндр
- `//walls <блок>` — создать стены вокруг выделенной области
- `//outline <блок>` — создать контур выделенной области

#### Операции с ландшафтом

- `//smooth` — сгладить ландшафт
- `//drain <радиус>` — удалить воду вокруг
- `//move <расстояние> [направление]` — переместить выделенную область

## Image On Map

Плагин для создания изображений на картах.

### Создание и управление изображениями

Основные команды:
- `/tomap <URL>` — создать карту с изображением по ссылке
- `/tomap <URL> <number>` — создать несколько карт для большого изображения
- `/maps` — управление созданными картами

Чтобы создать изображение на карте, найди ссылку на картинку (формат JPEG или PNG) и используй команду `/tomap <ссылка>`.

### Создание больших изображений

Для создания изображения размером больше одной карты, используй:
```
/tomap <ссылка> <количество-карт>
```

Например, `/tomap https://example.com/image.png 4` создаст изображение на 4 картах.

После создания изображения, оно появится в твоем инвентаре. Разместить карты на рамках нужно в правильном порядке — сначала верхний ряд слева направо, затем следующий ряд.

### Управление созданными картами

Команда `/maps` открывает меню управления твоими картами. В этом меню можно:
- Просматривать созданные изображения
- Удалять ненужные карты
- Получать дополнительные копии карт
