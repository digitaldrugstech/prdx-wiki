# Творческий сервер

Кроме выживания есть творческий сервер. Мир разделён на участки — проектируй постройки, чтобы потом перенести их в выживание

`/p auto` — занять случайный участок

## Axiom

Основной инструмент для строительства на креативе. Клиентский мод с мощным 3D-редактором прямо в игре — кисти, инструменты рельефа, работа с блоками, скульптинг

Из-за того что сервер пиратский, оригинальный Axiom не подключится. Используй нашу сборку:

## Плагины

<CollapseBlock>

<Collapse summary="PlotSquared — Управление участками">

#### Основные команды

| Команда | Описание |
|---------|----------|
| `/p help` | Все команды плагина |
| `/p auto` | Занять случайный участок |
| `/p claim` | Занять участок, на котором стоишь |
| `/p merge` | Объединить несколько участков рядом |
| `/p visit <игрок>` или `/p v` | Посетить участок игрока |
| `/p alias set <название>` | Установить название участка |
| `/p list <игрок>` | Список участков игрока |
| `/p info` | Информация о текущем участке |
| `/p middle` | Телепортироваться на середину участка |

#### Доступ к участку

| Команда | Описание |
|---------|----------|
| `/p trust <игрок>` | Дать игроку возможность строить |
| `/p deny <игрок>` | Запретить игроку находиться на участке |
| `/p deny *` | Запретить доступ всем |
| `/p allow <игрок>` | Разрешить доступ конкретному игроку |

#### Настройка участка

| Команда | Описание |
|---------|----------|
| `/p flag list` | Список доступных флагов |
| `/p flag add time <время>` | Установить время |
| `/p flag add pvp true` | Разрешить PVP |

</Collapse>

<Collapse summary="GoPaint & GoBrush — Кисти для ландшафтов">

GoPaint — возьми перо (`minecraft:feather`). :kbd[ЛКМ] — настройки, :kbd[ПКМ] — кисть

GoBrush — то же самое, но с кремнём (`minecraft:flint`)

`/gp`, `/gopaint` — команды GoPaint

`/gb`, `/gobrush` — команды GoBrush

<Collapse summary="Как перенести постройку с выживания на креатив">

На сервере с выживанием работают команды выделения WorldEdit: `//pos1`, `//pos2`, `//expand`

Выдели территорию, напиши `//copy` и `//schem save` — схематика сохранится

Схематики синхронизируются с креативом. Загрузи командой `//schem load` и вставь через `//paste`

</Collapse>
