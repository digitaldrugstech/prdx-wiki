# Бот Plasmo Sync

Этот бот автоматически синхронизирует:
- Ники игроков в Discord с никами в игре
- Роли в Discord с игровыми группами

## Преимущества использования

- Легко узнавать игроков по одинаковым никам
- Автоматическое обновление ролей при изменениях на сервере
- Возможность ограничить доступ к каналам только для игроков сервера

## Установка и настройка

### 1. Добавь бота на свой сервер

Боту потребуются права на управление ролями и никнеймами.

<FatLink href="https://discord.com/api/oauth2/authorize?client_id=944529811362181171&permissions=8&scope=bot%20applications.commands">
    Пригласить бота
</FatLink>
    
### 2. Настрой параметры синхронизации

Используй команду `/settings` для включения:
- Синхронизации ников
- Синхронизации ролей
- Автоматического вайтлиста

### 3. Подготовь роли для синхронизации

Создай нужные роли в Discord:
- Используй любые названия и цвета
- Можно синхронизировать все или только избранные роли

### 4. Установи правильную иерархию ролей

Роль бота `Plasmo Sync RP` должна быть выше всех синхронизируемых ролей.

<figure>
![](/assets/guides/sync/role_order.png)
:figcaption[Правильное расположение ролей]
</figure>

### 5. Настрой соответствие ролей
    
Используй команду `/set-role <роль_на_сервере> @discord_роль`:

Пример: `/setrole Игрок @[Synced] Player`

<figure>
![](/assets/guides/sync/set_role.jpeg)
:figcaption[Пример настройки роли]
</figure>
    
Для удаления соответствия используй: `/reset-role <роль>`

<CustomBlock type="warn">

#### Не забудь включить синхронизацию

Проверь, что синхронизация ролей активирована в настройках бота:

![](/assets/guides/sync/sync_settings.jpeg)

</CustomBlock>

### 6. Проверь настройки

Команда `/settings` показывает текущую конфигурацию бота.

### 7. Выполни первичную синхронизацию

Запусти команду `/everyone-sync`, чтобы обновить всех участников сервера.

<figure>
![](/assets/guides/sync/sync_process.jpeg)
:figcaption[Процесс синхронизации]
</figure>
    
Это нужно сделать только один раз. Процесс может занять от нескольких секунд до 2 минут из-за ограничений Discord.
    
<figure>
![](/assets/guides/sync/sync_done.jpeg)
:figcaption[Синхронизация завершена]
</figure>
