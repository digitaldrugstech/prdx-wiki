# Кастомные крафты

## Прозрачные рамки

Чтобы создать прозрачные рамки — обложите зелье невидимости из дыхания дракона (0:45) обычными рамками

### Яркость

Уровень света (0–15) настраивается через камнерез. Положи блок света в слот, выбери уровень. 15 = максимум, 0 = без света (но блокирует спавн мобов)

![Настройка яркости через камнерез](/assets/features/crafts/stonecutter-light.jpg)

### Размещение

- :kbd[ПКМ] по блоку — поставить
- Блок невидим для всех
- Возьми блок света в руку чтобы увидеть установленные — появятся частицы
- :kbd[ЛКМ] — сломать, выпадет с сохранением яркости
- Не имеет коллизии, не смывается водой
