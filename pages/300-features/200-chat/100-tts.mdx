# Озвучка локального чата

![](/assets/features/chat/tts.png)

Сообщения в локальном чате озвучиваются через Plasmo Voice. Пять голосов на русском, у каждого игрока свой

## Громкость

Не хочешь слышать озвучку — открой настройки Plasmo Voice (:kbd[V] → :kbd[Громкость]) и выкрути ползунок TTS на 0

![](/assets/features/chat/tts-volume.png)

Подробнее про настройку Plasmo Voice — [Голосовой чат](/wiki/mods/voice)

<Badge color="purple">Fusion</Badge>

Подписчики Fusion могут выбрать один из четырёх альтернативных голосов и настроить скорость речи через `/tts`

![](/assets/features/chat/tts-settings.png)
