# Чат общины в Discord

![](/assets/features/murteam/bridge.png)

Добавь бота на любой свой Discord-сервер и привяжи канал к общине. Сообщения синхронизируются в обе стороны — пишешь в Discord, появляется в игре, и наоборот

<FatLink href="https://discord.com/oauth2/authorize?client_id=1508047570176446595&permissions=8&integration_type=0&scope=bot+applications.commands">Добавить бота</FatLink>

## Команды

| Команда | Описание |
|---------|----------|
| `/murteam` | Панель управления — привязка, отвязка, настройки |
| `/online` | Кто из общины сейчас в игре |

![](/assets/features/murteam/panel.png)

## Как привязать

1. Пригласи бота на Discord-сервер общины
2. Напиши `/murteam` — откроется панель
3. Нажми «Привязать» и выбери общину из списка
4. Бот создаст вебхук в канале и начнёт синхронизацию

Один Discord-сервер — одна община. Привязать может только модератор или владелец общины

## Уведомления

Бот отправляет в канал системные события:

- Игрок из общины зашёл на сервер
- Игрок из общины вышел
- Игрок из общины умер

Каждый тип можно включить или выключить через панель `/murteam` → Настройки

![](/assets/features/murteam/settings.png)

## Как выглядят сообщения

Сообщения из игры приходят в Discord с ником и аватаркой игрока через вебхук. Сообщения из Discord отправляются в командный чат общины с ником из Discord
