# Отличия от ваниллы

Сервер максимально близок к ванилле, но с включённым экспериментом [Villager Trade Rebalance](https://minecraft.wiki/w/Villager_Trade_Rebalance)

## Villager Trade Rebalance

Торговля жителей зависит от биома. Это касается библиотекарей, оружейников, картографов и странствующего торговца

![Жители из разных биомов на сервере](/assets/features/diff/villager_trade_rebalance.jpg)

### Библиотекари

Книги зачарований привязаны к биому. У мастера-библиотекаря есть одна «специальная» книга с фиксированным уровнем, остальные книги «обычные» с рандомным уровнем

**Специальные книги (только мастер):**

| Биом | Зачарование |
|------|-------------|
| Пустыня | Efficiency III |
| Равнина | Protection III |
| Саванна | Sharpness III |
| Снежный | Silk Touch |
| Тайга | Fortune II |
| Джунгли | Unbreaking II |
| Болото | Mending |

**Обычные книги:**

| Биом | Зачарования |
|------|-------------|
| Пустыня | Fire Protection, Thorns, Infinity |
| Равнина | Punch, Smite, Bane of Arthropods |
| Саванна | Knockback, Curse of Binding, Sweeping Edge |
| Снежный | Aqua Affinity, Looting, Frost Walker |
| Тайга | Blast Protection, Fire Aspect, Flame |
| Джунгли | Feather Falling, Projectile Protection, Power |
| Болото | Depth Strider, Respiration, Curse of Vanishing |

Зачарования для арбалета, удочки, трезубца и булавы убраны из торговли

### Оружейники

Алмазная броня теперь стоит алмазы + изумруды. Ассортимент зависит от биома:

| Биом | Особенность |
|------|-------------|
| Джунгли, Болото | Только кольчужная броня |
| Саванна | Проклятая алмазная броня по сниженной цене |
| Тайга | Можно обменять одну алмазную деталь на другую (полный сет за 8 алмазов) |
| Снежный | Frost Walker ботинки, Aqua Affinity шлем |

### Картографы

Продают карты привязанные к своему биому: карты деревень, храмов в джунглях, хижин ведьм

### Странствующий торговец

Сниженные цены, расширенный ассортимент. Теперь может покупать предметы у игроков
