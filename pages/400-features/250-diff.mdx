# Отличия от ванильной игры

Мы внесли несколько изменений в стандартные механики для лучшей балансировки экономики сервера.

## Торговля с жителями

**Нет скидок у жителей**
- Не работает метод "зомбификации"
- Нет скидок за эффект "Герой деревни"

![](/assets/features/diff/villager.png)

## Тотемы бессмертия

**Рейды отключены**
- Тотемы можно получить только в лесных особняках

![](/assets/features/diff/raid.png)

## Шалкеры

**Размножение шалкеров невозможно**
- Но с каждого шалкера гарантированно выпадает 1 панцирь

![](/assets/features/diff/shulker.png)
