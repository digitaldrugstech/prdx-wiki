# Кастомные крафты

## Прозрачные рамки

Используй для декора невидимые рамки для предметов.

**Рецепт:**  
Обложи зелье невидимости (из дыхания дракона, 0:45) обычными рамками

![](/assets/features/crafts/frame.png)

## Невидимые источники света

Создавай невидимые источники освещения для скрытой подсветки строений.

**Рецепт:**  
Комбинируй оседающее зелье невидимости с любыми тремя светящимися блоками

![](/assets/features/crafts/light.png)
