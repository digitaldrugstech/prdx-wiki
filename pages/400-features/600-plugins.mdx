# Мелкие плагины

## Сидение на блоках

![](/assets/features/plugins/gsit.png)

Просто нажми ПКМ по ступеньке, ковру или полублоку, чтобы сесть на него.

## Мини-блоки и головы

У странствующего торговца можно купить:

- Мини-версии стандартных блоков (Vanilla Tweaks)
- Голову случайного игрока сервера

![](/assets/features/plugins/heads.png)

**Важно:** Головы игроков можно получить только у торговца, они редкие и ценные.

## Настройка стоек для брони

Деревянная мотыга позволяет редактировать положение стоек для брони — настраивай позы и создавай уникальные декорации.
