# Чат

## Локальный и глобальный режим

По умолчанию все сообщения отправляются в **локальный чат** (радиус 128 блоков, префикс **L**).

Для глобального чата добавь `!` перед сообщением: `!привет всем` (префикс **G**).

Игроки в Discord могут писать через канал `#игра` (префикс **D**).

## Полезные функции

### Координаты и предметы

Напиши `:loc:`, чтобы поделиться своим местоположением.

<figure>
![](/assets/features/chat/loc.png)
:figcaption[Эй, посмотри где я сейчас нахожусь :loc:]
</figure>

Покажи предмет в руке через `:item:`

### Метки и общины

Отправь метку: `:m1:` или `:м1:` (работает на русском и английском)

Отправь ссылку на общину: `:tНАЗВАНИЕ:`  
Например, [plasmorp.com/t/digitaldrugs](/t/digitaldrugs) → `:tdigitaldrugs:`

## Личные сообщения

Доступны даже между разными серверами (выживание/креатив).

### Команды

- `/w ник сообщение` — написать игроку (также `/msg`, `/tell`, `/m`)
- `/r сообщение` — ответить на последнее сообщение

## Веб-мессенджер

Доступен на [plasmorp.com/im](/im) для игроков с проходкой.

![](/assets/features/chat/messenger.png)

Синхронизирован с игрой:
- Можно писать офлайн-игрокам
- Получать сообщения, когда не в игре

## Настройки

Открой `/chat` для настройки:
- Сообщений о входе/выходе
- Глобального чата
- Сообщений о смерти
- Достижений
- Личных сообщений
- Звуковых уведомлений
- Фильтра матов

## Смена режима по умолчанию

Команда `/chat local/global/team` меняет канал, куда отправляются сообщения без префикса.

## Чат общин

Используй `@` для отправки в чат общины.

Выбери общину через `/teamschat название` или `/tc название`.  
Пример: `/tc beedrills`

## РП команды

- Локальные: `/me`, `/try`, `/do`
- Глобальные: `/gme`, `/gtry`, `/gdo`

**Пример использования:**  
`/gme вызывает интерпол на :loc:` — для вызова помощи
