# Начать игру

## Шаг 1 — Купите проходку

PRDX.SO — закрытый сервер. Чтобы попасть внутрь, нужна проходка.

<FatLink href="/access">Купить проходку</FatLink>

Проблемы с оплатой? Напишите в канал `#🕵ㆍподдержка` в [Discord](https://discord.gg/GdtKTgRybY).

## Шаг 2 — Установите моды

Играть можно без модов, но с ними опыт значительно лучше. Мы рекомендуем два:

- **🎙 Plasmo Voice** — голосовой чат, слышишь тех, кто рядом
- **✨ Emotecraft** — эмоции и анимации для персонажа

<FatLink href="/500-mods">Инструкция по установке</FatLink>

<CollapseBlock>

<Collapse summary="Какой лаунчер использовать?">

Подойдёт любой — лицензионный или пиратский. Актуальную версию игры смотри на [странице статуса](/stats).

</Collapse>

<Collapse summary="Можно ли играть с телефона или консоли (Bedrock)?">

Нет. Поддержка Bedrock временно недоступна. Сервер работает только на **Java Edition** для ПК.

</Collapse>

</CollapseBlock>

## Шаг 3 — Подключитесь к серверу

Добавьте один из адресов в список серверов Minecraft:

**Выживание**
```
prdx.so
```

**Творческий**
```
cr.prdx.so
```
