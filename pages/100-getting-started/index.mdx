# Начать игру

## 1. Купите проходку

<FatLink href="/access">Купить проходку</FatLink>

Если у вас есть проблемы с оплатой, то откройте тикет в канале `#🕵ㆍподдержка` на нашем [Дискорд сервере](https://discord.gg/prdxso)

## 2. Установите рекомендуемые моды

На сервере можно играть без модов, но мы рекомендуем скачать моды на голосовой чаты, эмоции и ресурспак сервера

- [Моды и ресурсы](/wiki/mods)

<CollapseBlock>

<Collapse summary="Какой можно использовать лаунчер?">

На сервер можно зайти с любого пиратского или лицензионного лаунчера

Версию игры можно узнать [тут](/stats)

</Collapse>

<Collapse summary="Как играть с Бэдрок (телефоны и консоли)">

Сервер работает на Java версии Майкрафт (ПК), но также поддерживает кросплатформенную игру с Bedrock издания (телефоны и консоли)

Учтите, что на Bedrock не будет работать мод на голосовой чат. Мы не рекомендуем покупать проходку на сервер если вы планируете играть только с Bedrock издания

- [Начать играть с Bedrock](/wiki/getting-started/bedrock)

</Collapse>

</CollapseBlock>

## 3. Подключитесь к серверу

Подключайтесь по адресу:

### Адрес сервера с выживанием

```
prdx.so
```

### Адрес творческого сервера

```
cr.prdx.so
```
