# Начать игру

**prdx.so** — приватный ванильный Minecraft сервер. Проходка даёт только доступ, никаких привилегий. Торговля, общины, законы — всё делают сами игроки. Каждый сезон — чистый мир

## Шаг 1 — Купи проходку

Сервер закрытый, нужна проходка

<FatLink href="/access">Купить проходку</FatLink>

## Шаг 2 — Прочитай правила

<FatLink href="/wiki/rules">Правила сервера</FatLink>

## Шаг 3 — Установи моды

Можно играть без модов, но с ними лучше

- **Plasmo Voice** — голосовой чат, слышишь тех кто рядом
- **Emotecraft** — эмоции и анимации для персонажа

<FatLink href="/wiki/mods">Инструкция по установке</FatLink>

<Collapse summary="Какой лаунчер использовать?">

Любой — лицензионный или пиратский. Актуальную версию игры смотри на [странице статуса](/stats)

</Collapse>

<Collapse summary="Можно ли играть с телефона или консоли?">

Нет. Сервер работает только на **Java Edition** для ПК

## Шаг 4 — Подключись к серверу

**Выживание** — Java Edition 1.21.11
```
survival.prdx.so
```

**Творческий**
```
creative.prdx.so
```
