# Начать игру

**prdx.so** — приватный ванильный Minecraft сервер с РП-уклоном. Никаких донаторских привилегий, никаких читов. Игроки сами строят города, ведут торговлю, объединяются в общины и принимают законы. Каждый сезон начинается с чистого мира

## Шаг 1 — Купи проходку

Сервер закрытый, нужна проходка

<FatLink href="/access">Купить проходку</FatLink>

Проблемы с оплатой? Напиши в `#🕵ㆍподдержка` в [Discord](https://discord.gg/GdtKTgRybY)

## Шаг 2 — Установи моды

Можно играть без модов, но с ними лучше

- **Plasmo Voice** — голосовой чат, слышишь тех кто рядом
- **Emotecraft** — эмоции и анимации для персонажа

<FatLink href="/mods">Инструкция по установке</FatLink>

<CollapseBlock>

<Collapse summary="Какой лаунчер использовать?">

Любой — лицензионный или пиратский. Актуальную версию игры смотри на [странице статуса](/stats)

</Collapse>

<Collapse summary="Можно ли играть с телефона или консоли?">

Нет. Сервер работает только на **Java Edition** для ПК

</Collapse>

</CollapseBlock>

## Шаг 3 — Подключись к серверу

**Выживание**
```
survival.prdx.so
```

**Творческий**
```
creative.prdx.so
```
