# Играть с Bedrock

## Важные отличия Bedrock-версии

На что обратить внимание:
- Ник в Xbox Live должен совпадать с ником на сервере
- Авторизация происходит через пароль, а не по ссылке
- Для подключения используется специальный адрес

## 1. Проверь свой ник

### Где найти Xbox-ник

Зайди в настройки Minecraft, вкладка **Профиль**. Сверху показан твой тег игрока.

<figure class="transparent">
![](/assets/getting-started/bedrock/xbox_nick.png)
</figure>

### Сверь с ником на сервере

Авторизуйся на сайте сервера и нажми на иконку в правом верхнем углу.
Ники должны совпадать!

<figure>
![](/assets/getting-started/bedrock/plasmo_nick.png)
:figcaption[Ник на сервере]
</figure>

### Если ники не совпадают

<CustomBlock>

#### Смена ника:

1. В Discord нажми ПКМ по иконке сервера
2. Выбери **Настр. профиль сервера**
3. Введи новый ник (соблюдай регистр!) и сохрани

Менять ник можно раз в 3 дня.

</CustomBlock>

<CustomBlock type="warn">

#### Важно!

После смены ника в Java-версии тоже придется использовать новый ник.

</CustomBlock>

## 2. Установи пароль

Для Bedrock-игроков используется авторизация через пароль.

<CustomBlock>

#### Установка пароля:

1. Перейди в личные сообщения с ботом: [Discord](https://discord.com/channels/@me/айди бота)
2. Напиши команду `!password <твой_пароль>`

</CustomBlock>

<figure>
![](/assets/getting-started/bedrock/password.png)
:figcaption[Успешная смена пароля]
</figure>

Для входа на сервер используй команду:
```
/login <твой_пароль>
```

<CustomBlock type="warn">

#### Обрати внимание

Хотя пароль менее надежен чем ссылка, вероятность взлома очень низкая.

Сбросить пароль можно командой `!reset`

</CustomBlock>

<figure>
![](/assets/getting-started/bedrock/password_reset.png)
:figcaption[Сброс пароля]
</figure>

## 3. Подключись к серверу

### Адрес для Bedrock:
```
-
```

Введи адрес, сохрани его и подключись.

<figure>
![](/assets/getting-started/bedrock/connect.png)
:figcaption[Подключение к серверу]
</figure>

## 4. Авторизуйся на сервере

При входе появится сообщение с просьбой перейти по ссылке.

<figure>
![](/assets/getting-started/bedrock/login.png)
:figcaption[Запрос авторизации]
</figure>

Вместо перехода по ссылке введи команду:
```
/login <твой_пароль>
```

<figure>
![](/assets/getting-started/bedrock/login_command.png)
:figcaption[Ввод команды авторизации]
</figure>

Готово! Теперь ты можешь играть на сервере.

<figure>
![](/assets/getting-started/bedrock/done.png)
:figcaption[Успешный вход]
</figure>
