# Играть с Bedrock

## Важные отличия Bedrock-версии

Три ключевых момента:
- Ник в Xbox Live должен совпадать с ником на сервере
- Для входа нужен пароль вместо ссылки
- Используется специальный адрес сервера

## 1. Проверь свой ник

### Где найти Xbox-ник

Открой настройки Minecraft, вкладка **Профиль**. Вверху указан твой тег игрока.

<figure class="transparent">
![](/assets/getting-started/bedrock/xbox_nick.png)
</figure>

### Сверь с ником на сервере

Авторизуйся на сайте и нажми на иконку профиля в правом верхнем углу.
Ники должны полностью совпадать!

<figure>
![](/assets/getting-started/bedrock/plasmo_nick.png)
:figcaption[Ник на сервере]
</figure>

### Если ники не совпадают

<CustomBlock>

#### Смена ника:

1. В Discord нажми ПКМ по иконке сервера
2. Выбери **Настр. профиль сервера**
3. Введи новый ник (соблюдай регистр!) и сохрани

Ограничение: смена ника доступна раз в 3 дня.

</CustomBlock>

<CustomBlock type="warn">

#### Важно!

После смены ника в Discord новый ник будет применяться и для Java-версии.

</CustomBlock>

## 2. Установи пароль

Bedrock-игроки используют авторизацию по паролю.

<CustomBlock>

#### Установка пароля:

1. Перейди в личные сообщения с ботом: [Discord](https://discord.com/channels/@me/айди бота)
2. Отправь команду `!password <придуманный_пароль>`

</CustomBlock>

<figure>
![](/assets/getting-started/bedrock/password.png)
:figcaption[Успешная установка пароля]
</figure>

Для входа на сервер используй:
```
/login <твой_пароль>
```

<CustomBlock type="warn">

#### Безопасность

Пароль менее надежен чем ссылка, но риск взлома минимален.

Сбросить пароль можно командой `!reset`

</CustomBlock>

<figure>
![](/assets/getting-started/bedrock/password_reset.png)
:figcaption[Сброс пароля]
</figure>

## 3. Подключись к серверу

### Адрес для Bedrock:
```
-
```

Введи адрес, добавь его в список серверов и подключись.

<figure>
![](/assets/getting-started/bedrock/connect.png)
:figcaption[Подключение к серверу]
</figure>

## 4. Авторизуйся на сервере

При входе увидишь сообщение с запросом авторизации.

<figure>
![](/assets/getting-started/bedrock/login.png)
:figcaption[Запрос авторизации]
</figure>

Вместо перехода по ссылке введи:
```
/login <твой_пароль>
```

<figure>
![](/assets/getting-started/bedrock/login_command.png)
:figcaption[Ввод команды авторизации]
</figure>

Готово! Теперь можешь играть на сервере.

<figure>
![](/assets/getting-started/bedrock/done.png)
:figcaption[Успешный вход]
</figure>
